# Soundwave

This repository contains the Soundwave mobile app (React Native / Expo) and backend (Node.js + Express + Supabase).

Quick start

Backend:

```bash
cd backend
npm install
cp .env.example .env   # fill in DB and Supabase keys
npm run dev
```

Mobile (Expo):

```bash
cd mobile
npm install
expo start --tunnel
```

See `SETUP.md` and `backend/SETUP.md` for more details.
